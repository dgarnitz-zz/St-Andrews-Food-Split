MyFlaskApp
Author: David A . Garnitz

Purpose: To facilitate food delivery order sharing for students in the library
or computer science labs.

Use Cases:
-Create a new order
-Add food to an existing order
-Look up a previous order

By default, all orders stay open for one hour.
